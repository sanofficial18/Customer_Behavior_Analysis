## Project Steps

1. **Data Loading**

   * Imported dataset into Python using Pandas.

2. **Exploratory Data Analysis (EDA)**

   * Checked data types, missing values, duplicates
   * Analyzed distributions, trends, and correlations
   * Created visualizations for better understanding

3. **Data Cleaning**

   * Handled missing values
   * Removed duplicates
   * Standardized column formats
   * Corrected data inconsistencies

4. **SQL Analysis**

   * Loaded cleaned data into SQL database
   * Performed analytical queries using:

     * Joins
     * Group By
     * Aggregations
     * Filtering
     * Subqueries

5. **Power BI Dashboard**

   * Created interactive dashboard with KPIs
   * Added filters and slicers
   * Designed user-friendly visuals

6. **Report Creation**

   * Documented insights, observations, and conclusions

7. **Presentation (PPT)**

   * Created professional presentation using **Gamma AI**
   * Converted insights into business-friendly slides

   ```
   pip install pandas numpy matplotlib seaborn
   ```
4. Run the notebook cells step-by-step
5. Import cleaned dataset into SQL database
6. Execute SQL queries provided
7. Open Power BI file to view dashboard
8. Review report and PPT for final insights
